튜토리얼

단계별 튜토리얼로 홍익 언어를 체계적으로 학습할 수 있습니다.

1. 첫 번째 프로그램 만들기

홍익 언어로 "안녕하세요!"를 출력하는 프로그램을 만들어보세요.

출력("안녕하세요!")
에디터에서 실습하기

2. 변수 사용하기

이름을 변수에 저장하고 출력해보세요.

이름 = "홍길동"
출력(이름)
에디터에서 실습하기

3. 조건문으로 분기하기

나이에 따라 다른 메시지를 출력해보세요.

나이 = 20
만약 나이 >= 20:
    출력("성인입니다.")
아니면:
    출력("미성년자입니다.")
에디터에서 실습하기

4. 반복문으로 반복하기

반복문을 사용해 0부터 4까지 숫자를 출력해보세요.

반복 i = 0 ~ 4:
    출력(i)
에디터에서 실습하기

5. 함수로 재사용하기

함수를 정의하고, 이름을 전달해 인사말을 출력해보세요.

함수 인사하기(이름):
    출력("안녕하세요, " + 이름 + "!")

인사하기("홍길동")
에디터에서 실습하기
