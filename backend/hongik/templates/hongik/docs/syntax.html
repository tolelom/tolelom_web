문법

홍익 언어의 기본 문법을 소개합니다.

1. 출력

:"안녕, 홍익!" 출력한다.

화면에 메시지를 출력합니다.

2. 변수

[문자]이름 = "홍길동"
:이름 출력한다.

변수를 선언하고 값을 저장할 수 있습니다.

3. 조건문

나이 = 20
만약 나이 >= 20:
    출력("성인입니다.")
아니면:
    출력("미성년자입니다.")

조건에 따라 다른 코드를 실행할 수 있습니다.

4. 반복문

반복 i = 0 ~ 4:
    출력(i)

특정 코드를 여러 번 반복해서 실행할 수 있습니다.

5. 함수

함수 인사하기(이름):
    출력("안녕하세요, " + 이름 + "!")

인사하기("홍길동")

함수를 정의하고, 필요할 때 호출할 수 있습니다.

6. 주석

# 이것은 주석입니다.

코드에 설명을 추가할 수 있습니다.

7. 자료형

홍익 언어는 다음과 같은 자료형을 지원합니다:

  • 숫자: 42, 3.14
  • 문자열: "홍익"
  • 불리언: 참, 거짓
  • 리스트: [1, 2, 3]
